# Security Policy

## Supported versions

VoidCode is still in early development. At this stage, only the `master` branch is considered supported for security fixes.

| Version | Supported |
| --- | --- |
| `master` | Yes |
| Released tags | No |

## Reporting a vulnerability

Please do **not** report security vulnerabilities in public issues or pull requests.

Use one of these private channels instead:

- GitHub Private Vulnerability Reporting, if it is enabled for this repository
- Email: `security@example.com`

When possible, include:

- a clear description of the issue
- affected components or paths
- reproduction steps or a proof of concept
- impact assessment
- any suggested mitigations

We will aim to acknowledge reports promptly, validate the issue, and coordinate a fix before public disclosure.
