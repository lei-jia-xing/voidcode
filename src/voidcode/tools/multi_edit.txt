Apply multiple edits to a file sequentially.

Usage:
- Use this tool when you need several ordered replacements in one existing file.
- The path parameter must be a workspace-relative file path.
- The edits parameter must be a non-empty array of objects shaped like {oldString, newString, replaceAll}.
- Do not use it for multiple files; use apply_patch for cross-file changes.
- Use the returned per-edit metadata and final diff as the source of truth before making follow-up edits.
