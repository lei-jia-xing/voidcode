- Search the web for external information and current references
- Returns ranked result titles, URLs, and snippets
- Use this tool when you need information beyond the local repository or do not yet know the exact URL

Usage notes:
  - The query parameter is required
  - numResults is optional and defaults to a small result set
  - Use web_fetch when the exact URL is already known
  - Treat external search results as research leads, not repository truth
