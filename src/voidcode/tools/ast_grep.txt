Use ast-grep tools for structural code matching when syntax shape matters more than literal text.

Usage:
- ast_grep_search performs structural search and returns AST matches.
- ast_grep_preview previews a rewrite without modifying files.
- ast_grep_replace applies a structural rewrite and requires apply=true.
- pattern is required. path must be workspace-relative. lang is optional but often improves correctness.
- Use preview before replace when the rewrite scope is not already obvious.
- Prefer returned match and replacement counts over prose summaries when planning the next step.
