Use this tool when you know the exact file you want to inspect inside the workspace.

It is best for reading UTF-8 text files directly, and it can also return images and PDFs as attachments. For text files, the result is line-numbered and starts from the top of the file by default.

If the file is large, continue reading it with a later offset instead of switching to shell commands. When you do not know the exact path yet, use `list` or `glob` first. When you need to search for text across a known file or directory target, use `grep` instead.

This tool does not read directories.
