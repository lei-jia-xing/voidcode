Performs exact string replacements in files.

Usage:
- Read the file first when you need context to choose a safe replacement.
- The path parameter must be a workspace-relative file path.
- oldString and newString must be strings. oldString must match the existing file content.
- Use this tool for targeted changes to existing files; use write_file or apply_patch when creating files or replacing a complete file.
- The edit will fail if oldString is not found in the file.
- The edit will fail if oldString matches multiple locations and replaceAll is not true.
- Use replaceAll for replacing every occurrence in the file, such as a local rename.
